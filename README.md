# neo4j-nodejs-ts-express
Hello world nodejs typescript express api

## Pre-requisites
nodejs and yarn

## How to run

## Install deps
```
yarn install
```

### Start in dev mode
```
yarn start:dev
```

### Test with curl
```
curl localhost:3333/movie
```
